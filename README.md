# wschat

Асинхронный WebSocket чат-сервер на Rust. Поддерживает несколько комнат, broadcast сообщений, встроенный веб-клиент. Запускается одной командой — браузер всё остальное.

## Возможности

- Несколько независимых комнат (создаются на лету: `/ws/roomname`)
- Broadcast сообщений всем участникам комнаты
- Система ников (команда `/nick Имя`)
- Список участников (`/who`)
- Встроенный HTML-клиент на корневом URL
- Нет внешних зависимостей, кроме tokio

## Установка

```bash
cargo install wschat
# или
cargo build --release
./target/release/wschat
```

## Запуск

```bash
# По умолчанию на порту 8080
wschat

# Другой порт
wschat --port 9090

# С логами
RUST_LOG=info wschat
```

## Подключение

- Откройте браузер: `http://localhost:8080` — встроенный клиент
- WebSocket: `ws://localhost:8080/ws/general` — комната "general"
- Своя комната: `ws://localhost:8080/ws/myroom`

## Протокол

Простые текстовые сообщения. Специальные команды начинаются с `/`:

| Команда | Описание |
|---------|----------|
| `/nick Имя` | Сменить ник |
| `/who` | Список участников комнаты |
| `/rooms` | Список активных комнат |
